Privacy Policy

隐私政策

最后更新日期：2026 年 3 月 11 日

1. 概述

福州堆栈科技有限公司（Fuzhou Stack Technology Co., Ltd.，以下简称"我们"）非常重视您的隐私保护。本隐私政策说明了我们在您使用 OpenAkita 产品和服务时如何收集、使用和保护您的信息。

2. 信息收集

OpenAkita 开源版（本地部署）：

  • OpenAkita 开源版在本地运行，所有对话数据、配置信息和记忆数据均存储在您的本地设备上。
  • 我们不会主动收集或上传您的对话内容、个人信息或使用数据。
  • 您与第三方 LLM 服务商（如 OpenAI、Anthropic、DeepSeek 等）的通信由相应服务商的隐私政策管辖。

OpenAkita 官方网站：

  • 访问日志 — 我们的网站托管服务可能记录基本的访问日志（IP 地址、浏览器类型、访问时间），用于安全防护和统计分析。
  • 下载统计 — 我们可能统计安装包的下载次数，用于了解产品使用趋势。

商业定制版：

  • 商业定制服务中涉及的数据收集和处理方式以双方签署的商业合同和数据处理协议为准。

3. 信息使用

我们收集的有限信息仅用于：

  • 改进产品功能和用户体验。
  • 保障服务安全和稳定运行。
  • 响应您的技术支持请求。
  • 履行法律义务。

我们不会将您的信息出售给任何第三方。

4. 第三方服务

OpenAkita 可能集成以下第三方服务，这些服务有各自的隐私政策：

  • LLM 服务商（OpenAI、Anthropic、DeepSeek 等）— 您的对话内容会发送给所选服务商处理。
  • IM 平台（Telegram、飞书等）— 通过 IM 通道交互时，消息经由相应平台传递。
  • GitHub — 开源代码托管和社区协作。

我们建议您查阅相关第三方服务的隐私政策。

5. 数据安全

我们采取合理的技术和管理措施保护您的信息安全，包括但不限于：

  • 本地数据存储架构 — 核心数据默认保存在您的设备上，不经过我们的服务器。
  • 传输加密 — 网站和 API 通信均使用 HTTPS 加密。
  • 最小权限原则 — 仅收集服务运行所必需的最少信息。

6. 您的权利

根据适用的法律法规，您享有以下权利：

  • 访问和获取我们持有的您的个人信息。
  • 要求更正不准确的个人信息。
  • 要求删除您的个人信息。
  • 撤回您此前给予的同意。

如需行使上述权利，请通过以下联系方式与我们联系。

7. 政策变更

我们保留随时更新本隐私政策的权利。修改后的政策将在本页面发布并标注更新日期。

8. 联系方式

如对本隐私政策有任何疑问或需要行使您的数据权利，请联系我们：

  • 邮箱：openakita@fzstack.com
