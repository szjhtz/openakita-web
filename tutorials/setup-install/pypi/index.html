方式二：PyPI CLI 安装

适合研发环境和自动化部署脚本，可作为 CI/CD 基础安装路径。

返回总览：安装教程总览

# 1) 创建虚拟环境 python -m venv venv # 2) 激活环境（Linux/macOS） source venv/bin/activate # 3) 激活环境（Windows PowerShell） # .\\venv\\Scripts\\activate # 4) 安装 openakita 与常用扩展 pip install openakita[all] # 5) 初始化配置文件 openakita init # 6) 启动交互模式 openakita

按需安装可选包：openakita[feishu]、openakita[whisper]、openakita[browser] 等。

启动验证

openakita openakita serve openakita run "创建一个带测试的 Python 计算器"
